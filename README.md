# Обработка изображений фильтром Собеля

Десктопное приложение на Python для выделения границ на изображениях с помощью оператора Собеля.  
Результаты сохраняются в локальную библиотеку SQLite с возможностью просмотра истории.

## Возможности

- Загрузка изображений популярных форматов (PNG, JPG, JPEG, BMP, TIFF)
- Применение фильтра Собеля для детекции границ
- Автоматическое сохранение обработанных изображений в папку `processed/`
- Ведение истории обработки (SQLite): исходный путь, обработанный путь, дата и время
- Просмотр библиотеки обработанных изображений в отдельном окне

## Библиотеки

- OpenCV (`cv2`)
- NumPy
- Tkinter (обычно входит в состав Python)
